# Purple Stark Theme for VS Code

A warm, paper-like theme with purple accents. Available in light and dark variants.

## Variants

### Purple Stark Light

A warm, cream-toned light theme with soft purple highlights.
![Stark Light](https://i.ibb.co/XZkd3mxx/Screenshot-2025-12-16-at-16-52-44.png)

### Purple Stark Dark

A cozy dark theme with warm undertones and purple accents.

![Stark Dark](https://i.ibb.co/YFSGNnL0/Screenshot-2025-12-16-at-16-52-25.png)

## Installation

### From VS Code Marketplace

1. Open VS Code
2. Go to Extensions (`Cmd+Shift+X` / `Ctrl+Shift+X`)
3. Search for "Purple Stark Theme"
4. Click **Install**
5. Press `Cmd+K Cmd+T` / `Ctrl+K Ctrl+T` to open the theme picker
6. Select **Purple Stark Light** or **Purple Stark Dark**

### Manual Installation

1. Copy this folder to your VS Code extensions directory:
   - **macOS**: `~/.vscode/extensions/stark-theme`
   - **Windows**: `%USERPROFILE%\.vscode\extensions\stark-theme`
   - **Linux**: `~/.vscode/extensions/stark-theme`
2. Restart VS Code
3. Open Command Palette (`Cmd+Shift+P` / `Ctrl+Shift+P`)
4. Type "Color Theme" and select **Preferences: Color Theme**
5. Choose **Purple Stark Light** or **Purple Stark Dark**

### From VSIX File

1. Download `stark-theme-1.0.0.vsix`
2. Open VS Code
3. Open Command Palette (`Cmd+Shift+P` / `Ctrl+Shift+P`)
4. Type "Install from VSIX" and select **Extensions: Install from VSIX...**
5. Select the downloaded `.vsix` file

## Features

- Warm, paper-like background colors
- Purple accent colors for UI elements and syntax highlighting
- Comprehensive token colors for all major languages
- Semantic highlighting support for enhanced code coloring
- Git decoration colors for file status
- Integrated terminal colors
- Full support for:
  - JavaScript/TypeScript
  - React/JSX/TSX
  - HTML/CSS/SCSS
  - JSON/YAML
  - Markdown
  - Python, Go, Rust, and more
